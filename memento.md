# Résumé des commandes git

## Commandes de tous les jours

* `git add`
* `git commit`
* `git status`
* `git log`

## Commandes qui modifient l'histoire

* `git rebase`
* `git commit --amend`
* `git reset`
* `git cherry-pick`

## Commandes sur le stash

* `git stash`
* `git stash list`
* `git stash pop`

## Commandes liées aux branches

* `git branch`
* `git merge`
* `git rebase`
